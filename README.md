# Codex Skills Backup

This repository backs up personal Codex skills from:

`~/.codex/skills`

Included skills:

- `ai-investment-research`
- `cc-investment-workflow`
- `vc-research-reader`

## Restore

On a new computer, copy the folders under `skills/` into:

`~/.codex/skills/`

For example:

```bash
mkdir -p ~/.codex/skills
cp -R skills/* ~/.codex/skills/
```

Then restart Codex if the skills do not appear immediately.
